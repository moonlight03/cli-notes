# Vim（编辑器）

## 模式速记
- 普通模式：默认，执行命令
- 插入模式：输入文本（普通模式按 `i`）
- 命令行模式：以 `:` `/` `?` 开头

## 光标移动
- `h j k l`：左下上右
- `0` / `$`：行首/行尾
- `gg` / `G`：首行/末行
- `:n` 或 `nG`：到第 n 行

## 查找与替换
- `/word`、`?word`：向下/向上查找
- `n` / `N`：下一个/上一个
- `:s/old/new/g`：当前行替换
- `:1,$s/old/new/gc`：全文替换并逐个确认
- `:noh`：关闭高亮

## 复制/删除/撤销
- `v`：进入可视选择
- `y` / `yy`：复制选中/复制整行
- `d` / `dd`：删除选中/删除整行
- `p`：粘贴
- `u`：撤销；`Ctrl+r`：重做

## 文件操作
- `:w` / `:w!`：保存/强制保存
- `:q` / `:q!`：退出/强制退出
- `:wq`：保存并退出

## 粘贴/缩进与其它
- `:set paste` / `:set nopaste`：粘贴模式开/关
- `>` / `<`：右/左缩进一次（对选区或动作）
- `gg=G`：格式化全文缩进

## 常见异常
- 出现 `.swp` 冲突：确认没有其它 Vim 正在编辑；或删除对应的 `.swp` 文件再打开。
